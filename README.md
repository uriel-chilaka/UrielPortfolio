## 🧩 Getting Started

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/uriel-chilaka/UrielPortfolio.git

# Navigate into the project folder
cd UrielPortfolio

# Install dependencies
npm install

# Start the development server
npm start
